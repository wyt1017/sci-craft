# sci-citation

Manage citations and references following Nature or Science journal standards.

## Features
- Journal-specific citation formatting (Nature numbered, Science numbered)
- Four attribution types: direct support, background, methodological, general reference
- Integrity checks: read verification, claim-support alignment, no citation dumping
- Export format support (EndNote .enw, RIS, Zotero RDF, BibTeX)

## Usage

> "Find supporting references for this claim about CRISPR off-target effects"

> "Format these citations for Nature journal"

> "Check if my citations properly support their claims"

## Triggers

- "supporting references", "Zotero RDF"
