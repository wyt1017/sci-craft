# sci-paper2ppt

Generate Chinese PPTX presentation decks from scientific papers — ideal for journal clubs, lab meetings, and paper presentations.

## Features

- 10–15 slide structured presentation from any research paper
- Chinese slides with English key terms for precision
- Figure-driven design with minimal bullet points (≤ 6 per slide)
- python-pptx script output for fully editable PPTX files
- Covers: Background → Methods → Results → Discussion → Take-home

## Usage

After installation, provide a paper and trigger the skill:

> "Convert this paper to PPT for journal club"

> "Make a 论文PPT from this paper"

> "Paper to slides for 论文汇报"

## Triggers

- "paper PPT", "journal club", "paper to slides", "paper presentation"
- "论文PPT", "论文汇报", "期刊俱乐部"
