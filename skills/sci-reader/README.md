# sci-reader

Bilingual paper reader with source anchors — convert academic papers into structured Markdown with paragraph-level Chinese translations and precise source markers.

## Features

- Section-by-section bilingual output (English original → Chinese translation)
- Source anchors (`[Section X.Y, Para N]`) for precise cross-referencing
- Figure grounding with `[Fig. N]` inline anchors
- Key terminology extraction with bilingual definitions
- Full preservation of equations, tables, and figure captions

## Usage

After installation, provide a paper and trigger the skill:

> "Read this paper with bilingual reader"

> "Generate a full markdown summary with 原文对照"

> "全文翻译 this paper with source anchors"

## Triggers

- "read paper", "paper summary", "full markdown"
- "bilingual reader", "原文对照", "全文翻译", "论文阅读"
