# sci-data

Generate Data Availability statements and repository plans that satisfy journal requirements and FAIR principles.

## Features

- Journal-aware Data Availability statements (adapts to Nature, Science, etc.)
- Repository Plan mapping each data type to the best archival destination
- FAIR Compliance Checklist (Findable, Accessible, Interoperable, Reusable)
- Supports datasets, code, models, and protocols

## Usage

After installation, describe your paper's data artifacts and trigger the skill:

> "Generate a data availability statement for my Nature paper"

> "Create a 数据声明 with FAIR check"

> "Help me plan repository deposition (数据可用性)"

## Triggers

- "data availability", "data statement", "FAIR metadata", "repository plan"
- "数据声明", "数据可用性"
