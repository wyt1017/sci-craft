# sci-writing

Draft academic manuscript sections that follow Nature or Science journal conventions.

## Features
- Section-aware drafting with structured move sequences for each manuscript part
- Journal configuration support (Nature: British English ≤30 words, Science: American English ≤25 words)
- Chinese-to-English draft translation that preserves intent, not clause order
- Hedging calibration matching claim strength to evidence
- Evidence ladder organization for Results (not chronological diary)

## Usage

> "Write an abstract for Nature journal about CRISPR-based gene therapy for sickle cell disease"

> "Draft the Introduction section for Science based on these notes: [paste Chinese notes]"

> "Help me write the Methods section following Nature conventions"

- "Nature writing", "Science writing", "draft paper"
