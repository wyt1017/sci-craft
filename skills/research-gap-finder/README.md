# research-gap-finder

通过系统性分析文献，使用五维空白矩阵识别研究空白，分析矛盾发现并挖掘潜在的未解决问题。

## Features
- 五维研究空白识别矩阵（理论、方法、情境、时间、整合空白）
- 矛盾与不一致性分析与溯源
- 空白重要性、新颖性与优先级评估
- 隐性空白挖掘（被忽视的声音、被低估的关系）
- 空白验证与文献再确认

## Usage

> "分析这个领域的研究空白和未解决的问题"

> "识别深度学习在医学影像领域的矛盾发现和空白"

## Triggers
- "research gap", "研究空白", "gap analysis", "空白分析"
- "contradiction analysis", "矛盾发现", "identify gap", "识别空白"
