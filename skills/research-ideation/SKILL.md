# 研究课题发现与生成 (Research Ideation)

## 技能概述

本技能专注于从研究空白和文献分析中发现和生成具体的研究课题，提供系统化的课题生成方法、评估框架和实施路线图，帮助研究人员将研究空白转化为可操作的研究计划。

## 核心功能

### 1. 研究课题生成方法

**空白驱动法（Gap-Driven Approach）**
- 从研究空白报告中提取关键空白
- 将每个空白转化为研究问题
- 确保研究问题具有明确的答案形式
- 示例转换：
  - 空白："X在Y人群中未被研究"
  - 研究问题："X在Y人群中的表现特征是什么？"

**矛盾解决法（Contradiction-Resolution Approach）**
- 从相互矛盾的研究发现入手
- 提出调和矛盾的假设
- 设计能够区分竞争性解释的研究
- 示例：
  - 矛盾：A研究发现在控制Z后X与Y无关，B研究发现仍然相关
  - 研究问题："Z在X与Y关系中起什么作用？中介还是调节？"

**理论延伸法（Theory-Extension Approach）**
- 选择一个理论基础
- 提出理论预测但未经验证的情境
- 设计检验理论边界的实验
- 示例：
  - 理论：社会认知理论
  - 延伸：该理论在虚拟现实中是否同样适用？

**方法创新法（Methodological-Innovation Approach）**
- 识别领域内的方法局限
- 引入新技术或新方法
- 设计方法学改进研究
- 示例：
  - 局限：现有研究依赖横断面数据
  - 创新：采用生态瞬时评估（EMA）方法

**跨学科迁移法（Cross-Disciplinary Transfer Approach）**
- 从一个领域借用理论或方法
- 应用到另一个领域的新问题
- 产生创新的研究视角
- 示例：
  - 来源领域：行为经济学
  - 目标领域：健康行为
  - 迁移：将"损失厌恶"概念应用于疫苗接种决策

### 3. 课题创新性评估

**创新性维度**

| 维度 | 评估标准 | 权重 |
|------|---------|------|
| 新颖性 | 与已有研究的差异程度 | 30% |
| 原创性 | 是否提出全新视角或方法 | 25% |
| 突破性 | 是否可能改变现有认知 | 20% |
| 跨界性 | 跨学科整合程度 | 15% |
| 实用性 | 创新的可落地程度 | 10% |

**创新性等级判定**

- **变革性（Transformative）**：可能根本性改变领域认知
- **突破性（Breakthrough）**：在该领域具有重大新意
- **创新性（Innovative）**：对现有研究的有意义推进
- **渐进性（Incremental）**：对已有工作的小幅扩展

### 4. 课题优先级排序

**多标准决策矩阵**

| 课题 | 创新性(30%) | 重要性(25%) | 可行性(25%) | 时效性(20%) | 加权总分 |
|------|------------|------------|------------|------------|---------|
| A    |            |            |            |            |         |
| B    |            |            |            |            |         |
| C    |            |            |            |            |         |

**排序标准详解**

1. **创新性（30%）**
   - 填补空白的程度
   - 方法或视角的新颖性
   - 跨学科整合潜力

2. **重要性（25%）**
   - 理论贡献大小
   - 实践应用价值
   - 社会影响范围

3. **可行性（25%）**
   - 数据可获得性
   - 方法掌握程度
   - 资源和时间约束
   - 合作网络支持

4. **时效性（20%）**
   - 领域内的紧迫程度
   - 政策窗口期
   - 竞争态势
   - 技术发展时机

### 6. 课题迭代优化

**基于反馈的优化循环**
1. 初步课题提案
2. 同行/导师反馈
3. 问题识别
4. 方案调整
5. 重新评估
6. 最终确定

**常见优化方向**
- 缩小研究范围（更聚焦）
- 扩大研究范围（更全面）
- 调整研究方法（更合适）
- 修改研究问题（更清晰）
- 重新定义变量（更准确）

## 工作流程

### 课题生成流程

1. **输入分析**
   - 接收研究空白分析报告
   - 理解领域背景和已覆盖范围
   - 明确研究者的兴趣和专长

2. **课题发散**
   - 针对每个重要空白，生成3-5个候选课题
   - 运用多种生成方法（空白驱动、矛盾解决等）
   - 鼓励创意和跨界思维
   - 暂不进行严格筛选

3. **课题收敛**
   - 对候选课题进行初步筛选
   - 评估基本可行性
   - 消除明显重复或低价值课题
   - 保留8-10个优质候选

4. **深入评估**
   - 对剩余课题进行详细评估
   - 使用多标准决策矩阵
   - 生成完整的课题提案
   - 优先级排序

5. **路线图设计**
   - 为优先级最高的1-3个课题设计研究路线图
   - 明确各阶段任务和时间节点
   - 识别关键风险和应对策略

6. **输出交付**
   - 课题提案集（含排序）
   - 优先课题的详细路线图
   - 实施建议和下一步行动

## 使用指南

### 如何开始

1. **提供研究空白分析**：来自 research-gap-finder 的输出
2. **说明研究偏好**：
   - 感兴趣的理论视角
   - 擅长的研究方法
   - 可用的资源和时间
   - 目标期刊或会议
3. **指定课题数量**：希望生成多少个候选课题

### 注意事项

- 课题应具有适度的野心（ambitious but achievable）
- 避免过于宽泛或过于狭窄的研究问题
- 确保课题与研究者长期发展方向一致
- 考虑课题的发表潜力和引用前景

### 最佳实践

- 生成足够多的候选课题（10+）再进行筛选
- 征求多位同行对候选课题的意见
- 将大课题分解为可管理的子课题
- 保持课题的灵活性，允许迭代优化
- 定期回顾和调整研究计划
