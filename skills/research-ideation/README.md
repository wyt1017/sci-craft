# research-ideation

基于研究空白分析，生成具有创新性和可行性的研究课题提案，提供优先级排序和研究路线图建议。

## Features
- 多种课题生成方法（空白驱动、矛盾解决、理论延伸、方法创新、跨学科迁移）
- 标准课题提案结构模板
- 课题创新性评估与等级判定
- 多标准决策矩阵优先级排序
- 研究路线图设计与迭代优化

## Usage

> "基于已有的空白分析，帮我生成研究课题"

> "为这个领域生成3-5个创新性研究选题并排序"

## Triggers
- "research ideation", "研究课题", "课题生成", "research topic"
- "研究选题", "research proposal", "课题提案", "generate ideas", "生成研究想法"
