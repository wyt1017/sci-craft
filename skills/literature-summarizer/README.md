# literature-summarizer

对已获取的学术文献进行系统性阅读、多层级摘要提取和知识图谱构建。

## Features
- 四层分层摘要生成（核心贡献→结构化摘要→详细解读→批判性评价）
- 文献精读笔记与结构化模板
- 知识图谱构建（概念提取、研究流派、方法论地图）
- 跨文献综合（主题聚类、证据强度评估、共识与争议识别）
- 文献矩阵表与可视化呈现

## Usage

> "帮我阅读并总结这篇文献的核心内容"

> "对这组文献进行综合阅读和跨文献对比分析"

## Triggers
- "literature summary", "文献摘要", "文献总结", "paper summary"
- "阅读文献", "reading notes", "阅读笔记", "knowledge graph", "知识图谱"
