# literature-search

学术研究文献检索与收集，帮助用户制定搜索策略、构建检索式、获取和管理文献。

## Features
- 研究问题转化为可检索策略（PICO/SPIDER框架）
- 关键词提取、扩展与检索式构建
- 多数据库检索（PubMed、Web of Science、Scopus等）
- 文献筛选与PRISMA流程管理
- 引文追溯与高级检索技巧

## Usage

> "帮我检索关于深度学习在医学影像中应用的文献"

> "制定一个关于机器学习辅助诊断的系统检索策略"
