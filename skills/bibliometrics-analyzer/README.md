# bibliometrics-analyzer

对学术文献进行系统的定量分析，通过文献计量学方法揭示研究领域的结构特征、发展趋势和知识基础。

## Features
- 共现分析与关键词集群识别
- 引用网络分析（文献共被引、期刊共被引、作者共被引）
- 突现检测与趋势分析
- 合作网络分析（作者、机构、国际合作）
- 影响力评估与可视化建议

## Usage

> "对这个领域的文献进行文献计量分析"

> "分析深度学习在医学影像领域的研究趋势和共现网络"

## Triggers

- "co-occurrence analysis", "共现分析", "research trend", "研究趋势"
