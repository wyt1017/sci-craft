# 文献计量分析 (Bibliometrics Analyzer)

## 技能概述

本技能专注于对学术文献进行系统的定量分析，通过文献计量学方法揭示研究领域的结构特征、发展趋势和知识基础，为研究决策提供数据支持。

## 核心功能

### 1. 文献计量基础分析

**基本统计指标**
- 年度发文量趋势
- 作者数量和合作网络
- 机构和国家分布
- 期刊分布和影响力
- 文献类型分布
- 语言分布
- 研究主题分布

**核心文献识别**
- 最高被引文献
- 近期高增长文献
- 关键转折点文献
- 奠基性文献
- 桥梁文献（连接不同研究群体）

### 2. 共现分析（Co-occurrence Analysis）

**关键词共现**
- 提取每篇文献的关键词
- 构建关键词共现矩阵
- 识别高频关键词
- 发现关键词集群（研究主题）
- 追踪关键词的时间演变

**分析指标**
- 关键词频次
- 关键词中心性（Centrality）
- 关键词突现值（Burst Score）
- 关键词时间线（Timeline View）

**关键词集群识别**
```
集群1：深度学习（中心性：0.45）
  - 卷积神经网络
  - 递归神经网络
  - 注意力机制
  
集群2：医学影像（中心性：0.38）
  - CT扫描
  - MRI
  - 图像分割
  
集群3：目标检测（中心性：0.32）
  - YOLO
  - Faster R-CNN
  - 边界框回归
```

### 3. 引用网络分析

**文献共被引分析（Bibliographic Coupling）**
- 两篇文献如果被同一篇后续文献引用，则形成共被引关系
- 识别研究前沿（Research Fronts）
- 发现相似的研究论文集群

**期刊共被引分析**
- 期刊之间的引用关系
- 识别期刊集群（研究领域）
- 发现期刊间的桥梁关系

**作者共被引分析**
- 作者之间的学术影响关系
- 识别学术共同体
- 发现关键学者

**引文网络可视化元素**
- 节点大小：文献/作者/期刊的影响力
- 节点颜色：所属集群/时间段
- 连线粗细：引用强度
- 布局算法：Force-directed、Spring embedding

### 4. 突现检测（Burst Detection）

**关键词突现**
- 识别突然增加频率的关键词
- 反映新兴研究热点
- 示例时间线：
  ```
  2018-2020: "transfer learning" 突现
  2020-2022: "self-supervised learning" 突现
  2022-至今: "foundation model" 突现
  ```

**文献突现**
- 识别突然被大量引用的文献
- 可能是突破性发现或方法创新

**期刊突现**
- 识别快速崛起的期刊
- 反映研究重心的转移

**突现强度计算**
- 突现起始时间
- 突现持续时间
- 突现强度（Burst Strength）
- 当前是否仍在突现中

### 5. 合作网络分析

**作者合作网络**
- 识别核心作者群
- 分析合作密度
- 发现孤立研究者
- 追踪合作模式演变

**机构合作网络**
- 识别领先机构
- 分析跨机构合作
- 发现合作联盟
- 评估机构影响力

**国际合作网络**
- 国家/地区合作模式
- 南北合作 vs 南南合作
- 识别国际合作热点
- 分析语言和文化因素影响

**合作网络指标**
- 网络密度（Network Density）
- 平均路径长度（Average Path Length）
- 聚类系数（Clustering Coefficient）
- 中心性指标（Betweenness Centrality）

### 6. 时间与趋势分析

**年度趋势分析**
- 发文量时间序列
- 主题演变时间线
- 引用影响力变化
- 方法学发展趋势

**研究阶段划分**
- 萌芽期：低发文量、探索性研究
- 成长期：快速增加的发文量
- 成熟期：稳定的发文量和主题
- 转型期：新方法的引入、范式转变

**研究生命周期识别**
- 新技术的生命周期曲线
- 识别技术的采纳阶段
- 预测未来的发展趋势

### 7. 影响力分析

**文献影响力**
- 总被引次数
- 年均被引率
- h指数（针对作者）
- 立即指数（Immediacy Index）

**期刊影响力**
- 影响因子（Impact Factor）
- CiteScore
- 分区（JCR Quartile）
- 即时指数

**作者影响力**
- 总引用数
- h指数
- g指数
- 领域相对影响力（Field-Weighted Citation Impact）

**可视化建议**
- 影响力热力图
- 影响力-时间散点图
- 影响力排名柱状图

## 数据来源

### 主要文献计量数据源

| 数据源 | 覆盖范围 | 计量工具 | 特点 |
|--------|---------|---------|------|
| Web of Science | 多学科 | CiteSpace、VOSviewer、BibExcel | 引文数据完整 |
| Scopus | 最广泛 | Scopus Analysis、VOSviewer | 覆盖期刊最多 |
| PubMed | 生物医学 | Pubmed、R Bibitem | 免费、医学专用 |
| Google Scholar | 最广泛 | 手动导出、Scholarly | 覆盖面广但不精确 |
| Dimensions | 多学科 | Dimensions Analytics | 包含专利和基金 |
| OpenAlex | 开放数据 | 自定义分析 | 免费开放 |

### 常用文献计量工具

**可视化工具**
- VOSviewer：简单易用、适合初学者
- CiteSpace：功能强大、中国学者开发
- Gephi：通用网络可视化工具
- Pajek：大型网络分析
- Bibliometrix/R包：R语言生态

**数据分析工具**
- R（bibliometrix、bibliography包）
- Python（scholar、pymetrics）
- Python（networkx、igraph）
- MATLAB

## 工作流程

### 标准文献计量分析流程

1. **数据获取**
   - 确定检索策略
   - 从数据库导出数据
   - 数据清洗和标准化
   - 确认数据质量

2. **基础分析**
   - 描述性统计分析
   - 核心文献识别
   - 初步趋势观察

3. **深度分析**
   - 共现分析
   - 引用网络分析
   - 突现检测
   - 合作网络分析

4. **结果解释**
   - 识别模式和规律
   - 发现异常和有趣现象
   - 与领域知识结合解读

5. **可视化呈现**
   - 选择合适的图表类型
   - 设计清晰的视觉呈现
   - 添加必要的注释

6. **报告生成**
   - 结构化呈现分析结果
   - 提供可操作的洞察
   - 指出分析局限性

### 数据清洗要点

```markdown
## 关键词标准化
- 统一拼写（colour/color）
- 合并同义词（AI/artificial intelligence）
- 拆分复合词（machine-learning/machine learning）
- 去除无意义词（study、research等）

## 作者名标准化
- 统一姓名列格式
- 处理缩写和全称
- 识别同名不同人
- 使用ORCID标识

## 机构名标准化
- 统一中英文名称
- 处理缩写和全称
- 识别机构合并和更名
```

## 使用指南

### 如何开始

1. **提供分析主题**：明确的研究领域或主题
2. **指定数据源**：可用的数据库和导出格式
3. **选择分析深度**：
   - 基础分析：仅描述性统计
   - 标准分析：基础+共现+引用
   - 深度分析：全部分析类型

### 注意事项

- 文献计量结果需要领域知识进行合理解读
- 数据质量直接影响分析结果
- 不同数据库的结果可能有差异
- 避免过度解读统计相关性

### 最佳实践

- 始终进行数据清洗和标准化
- 使用多种分析方法交叉验证
- 结合定性分析理解定量结果
- 定期更新分析以跟踪最新趋势
- 分享分析方法和数据以提高透明度
