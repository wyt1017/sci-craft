# sci-reviewer

Generate Nature/Science-style simulated peer review reports from three expert perspectives.

## Features
- Three independent referee reports (Methodology / Novelty-Significance / Presentation-Reproducibility)
- Structured report format: Summary, Major Concerns, Minor Concerns, Recommendation
- Cross-review synthesis identifying consensus concerns and conflicting opinions
- Integrated overclaim detection throughout all reports
- Journal-aware: follows {{name}} review standards

## Usage

> "Review this paper for Nature: [paste manuscript]"

> "Pre-submission review of this manuscript"

## Triggers
- "review paper", "peer review", "reviewer report", "pre-submission review"
- "Nature reviewer", "审稿", "审稿人视角"
