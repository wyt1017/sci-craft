# sci-response

Craft professional point-by-point response letters to reviewer comments following journal conventions.

## Features
- Comment triage: categorize each reviewer comment (factual / interpretation / methodological / editorial)
- Action mapping: map comments to specific actions (revise / add / experiment / decline with justification)
- Risk assessment: flag high-risk comments where disagreement is likely
- Structured response format per comment: Reviewer comment → Response → Manuscript change
- Tone guardrails: respectful, evidence-based, never defensive
- Integrated hedging and overclaim detection

## Usage

> "Help me respond to these reviewer comments for Nature: [paste comments]"

> "Write a rebuttal letter for this major revision"

## Triggers
- "response to reviewers", "rebuttal letter", "major revision"
- "reviewer response", "审稿回复", "回复审稿人"
