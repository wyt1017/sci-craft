# 系统综述支持 (Systematic Review)

## 技能概述

本技能专注于系统综述（Systematic Review）的制作全过程，遵循国际公认的标准和指南（如PRISMA、Cochrane），帮助研究人员制作高质量、可发表的系统综述。

### 3. 质量评估工具

**定量研究质量评估**

| 研究设计 | 推荐工具 | 评估维度 |
|---------|---------|---------|
| 随机对照试验（RCT） | Cochrane RoB 2 | 随机化、偏离干预、缺失数据、效应测量、选择性报告 |
| 非随机干预研究 | RoB 2.0 | 混淆、选择、分类、数据缺失、测量 |
| 队列研究 | Newcastle-Ottawa Scale | 选择、可比性、结局 |
| 病例对照研究 | Newcastle-Ottawa Scale | 选择、可比性、暴露 |
| 横断面研究 | JBI Critical Appraisal | 样本代表性、样本量、 confounder控制 |
| 诊断准确性研究 | QUADAS-2 | 患者选择、干预、参考标准、流程和时间 |

**定性研究质量评估**
- CASP Qualitative Checklist
- JBI Critical Appraisal Checklist for Qualitative Research
- COREQ（报告质量，非评估质量）

**混合方法研究**
- MMAT（Mixed Methods Appraisal Tool）
- JBI Mixed Methods Appraisal Tool

**系统综述/Meta分析本身的质量**
- AMSTAR 2（A MeaSurement Tool to Assess systematic Reviews）
- PRISMA-A（for abstracts）

### 4. GRADE证据质量分级

**证据质量等级**

| 等级 | 符号 | 含义 |
|------|------|------|
| 高 | ⊕⊕⊕⊕ | 非常确定真实效应接近真实效应 |
| 中 | ⊕⊕⊕○ | 真实效应可能接近真实效应 |
| 低 | ⊕⊕○○ | 真实效应可能远离真实效应 |
| 极低 | ⊕○○○ | 真实效应很可能远离真实效应 |

**降级因素**
1. **偏倚风险**：纳入研究存在方法学缺陷
2. **不一致性**：研究间结果高度异质
3. **间接性**：证据不能直接回答问题（PICO差异）
4. **不精确性**：置信区间宽或样本量小
5. **发表偏倚**：可能存在未发表的研究

**升级因素**（仅适用于观察性研究）
1. **效应量大**：相对危险度>2或<0.5
2. **剂量反应关系**：存在梯度效应
3. **残余混杂**：可能降低而非夸大效应

### 5. Meta分析支持

**异质性检验**
- Q统计量（Cochran's Q）
- I²统计量：
  - 0-40%：可能不重要
  - 30-60%：中度异质
  - 50-90%：较大异质
  - 75-100%：极度异质
- Tau²：估计研究间变异

**效应量选择**
- 连续变量：均差（MD）、标准化均差（SMD/Hedges' g）
- 二分类变量：风险比（RR）、比值比（OR）、风险差（RD）
- 生存数据： hazards ratio（HR）
- 相关系数：Pearson's r

**模型选择**
- 固定效应模型：假设所有研究估计同一效应
- 随机效应模型：假设效应量服从分布
- 选择依据：异质性程度和研究目的

**亚组分析**
- 按研究特征分组
- 按研究对象分组
- 按干预特征分组
- 注意：避免数据驱动的亚组分析

**敏感性分析**
- 排除高风险偏倚研究
- 使用不同效应模型
- 排除小样本研究
- 替代结局指标

**发表偏倚评估**
- 漏斗图（Funnel Plot）
- Egger's检验
- Begg's检验
- 失安全系数（Fail-safe N）

## 工作流程

### 系统综述标准流程

1. **准备阶段**
   - 确定研究问题（PICO）
   - 进行初步文献检索
   - 撰写研究方案
   - 在PROSPERO注册
   - 组建研究团队（至少2人独立工作）

2. **检索阶段**
   - 制定完整检索策略
   - 在多数据库执行检索
   - 补充检索（参考文献、灰色文献）
   - 导出和管理文献

3. **筛选阶段**
   - 去重
   - 标题和摘要初筛
   - 全文复筛
   - 记录排除原因
   - 生成PRISMA流程图

4. **数据提取阶段**
   - 设计数据提取表
   - 双人独立提取
   - 解决分歧
   - 联系原作者补充数据（如需要）

5. **质量评估阶段**
   - 选择适当的评估工具
   - 双人独立评估
   - 记录评估结果
   - 进行偏倚风险评估

6. **数据合成阶段**
   - 判断是否适合Meta分析
   - 选择效应量和统计模型
   - 执行Meta分析
   - 评估异质性
   - 进行亚组和敏感性分析
   - 评估发表偏倚
   - GRADE证据质量分级

7. **撰写阶段**
   - 按照PRISMA清单撰写
   - 绘制PRISMA流程图
   - 制作森林图等可视化
   - 内部审核和修改

8. **投稿阶段**
   - 选择目标期刊
   - 准备投稿材料
   - 响应审稿意见

## 使用指南

### 如何开始

1. **提供研究主题**：明确的研究问题
2. **说明综述类型**：系统综述、Meta分析、范围综述等
3. **指定进度**：
   - 从零开始：需要全流程指导
   - 进行中：需要特定环节的帮助
   - 已完成：需要写作和格式支持

### 注意事项

- 系统综述是一项耗时的工作，通常需要6-18个月
- 必须有至少两名研究者独立工作
- 研究方案应在开始前确定并注册
- 严格遵循PRISMA指南
- 保持透明和可重复

### 最佳实践

- 使用文献管理工具（EndNote、Zotero）管理文献
- 使用专门的系统综述软件（RevMan、Stata、R metafor）
- 定期与团队沟通进展
- 预注册研究方案
- 遵循报告指南（PRISMA、MOCHA、SPIDER）
- 考虑使用GRADEpro制作证据概况表
