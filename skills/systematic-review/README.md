# systematic-review

遵循PRISMA指南进行系统综述的全流程管理，包括研究方案制定、文献筛选、质量评估和数据提取。

## Features
- 研究方案制定与PROSPERO注册准备
- PRISMA 2020流程图与检查清单管理
- 多种质量评估工具（Cochrane RoB 2、NOS、QUADAS-2等）
- GRADE证据质量分级
- Meta分析支持（异质性检验、效应量选择、亚组与敏感性分析）

## Usage

> "帮我制作一个关于X干预对Y影响的系统综述"

> "设计一个PRISMA流程并进行Meta分析"
